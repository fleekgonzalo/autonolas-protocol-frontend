## Installation and Setup Instructions

Clone down this repository. You will need `node` version ">=14" and `yarn` version ">=1" installed globally on your machine.  

Installation:

`yarn`  

To Run Test Suite:  

`yarn testc`  

To Start Server:

`yarn dev`  

To Visit App:

`localhost:3000`
